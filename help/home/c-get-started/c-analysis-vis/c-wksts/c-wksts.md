# Kalkylblad{#worksheets}

{{eol}}

Data Workbench innehåller kalkylblad för att skapa ad hoc-filter och -mått för den aktuella visualiseringen.

Ett kalkylblad liknar ett vanligt kalkylblad, men till skillnad från ett kalkylblad kan alla celler som innehåller en formel definiera ett mätvärde.

![](assets/vis_Worksheet_TextAndFormula.png)

Till skillnad från mätvärden som skapats med [!DNL Metric Editor]visas inte mätvärden som definierats med kalkylblad i måttmenyerna, utan de kan sparas och inkluderas i alla arbetsytor.

Mer information om [!DNL Metric Editor], se [Skapa och redigera härledda mått](../../../../home/c-get-started/c-admin-intrf/c-prof-mgr/c-drvd-mtrcs.md#concept-e41723b342a849309874b26232224a40).
