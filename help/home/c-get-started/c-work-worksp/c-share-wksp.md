# Dela en arbetsyta{#share-a-workspace}

{{eol}}

Du kan använda arbetsytemenyn Exportera för att skriva ut och exportera arbetsytor till PNG-filer eller till Excel.

Klicka på **[!UICONTROL Export]**.
